Options Black-Scholes Call

C = S·N(d1) - K·e^(-rT)·N(d2) d1 = [ln(S/K) + (r + σ²/2)T] / σ√T d2 = d1 - σ√T

S = spot, K = strike, r = taux sans risque, σ = vol implicite, T = maturité, N() = loi normale cumulée

Options Approximation ATM

Call ATM ≈ 0.4 × S × σ × √T Straddle ATM ≈ 0.8 × S × σ × √T

Valable uniquement ATM (S ≈ K). Incontournable en entretien pour estimer rapidement un prix d'option.

Options Parité Put-Call

C - P = S·e^(-qT) - K·e^(-rT) ou simplement : C - P = F - K·e^(-rT)

q = taux de dividende. Si C-P ≠ F-Ke^(-rT), il existe un arbitrage.

Greeks Greeks principaux

Delta (Δ) = ∂C/∂S ≈ N(d1) Gamma (Γ) = ∂²C/∂S² = N'(d1)/Sσ√T Vega (ν) = ∂C/∂σ = S√T·N'(d1) Theta (Θ) = ∂C/∂t Rho (ρ) = ∂C/∂r

Delta : entre 0 et 1 pour un call. Gamma maximal ATM. Vega identique call et put même strike.

Forward Prix Forward

F = S · e^((r - q) · T) Avec dividendes discrets : F = (S - PV(div)) · e^(r·T)

r = taux sans risque, q = taux de dividende continu. Le Vega d'un forward est toujours nul.

Taux Taux Forward

(1 + r2)^T2 = (1 + r1)^T1 × (1 + f)^(T2-T1) En continu : f = (r2·T2 - r1·T1) / (T2 - T1)

f = taux forward entre T1 et T2. Si la courbe est croissante, les taux forwards sont supérieurs aux taux spots.

Vol Annualisation de la volatilité

σ_annuelle = σ_journalière × √252 σ_journalière = σ_annuelle / √252 Règle : vol mensuelle × √12 = vol annuelle

252 = nombre de jours de trading par an. Une action avec 1%/jour de vol a ~16% de vol annuelle.

Portfolio Variance d'un portefeuille 2 actifs

σ²_p = w1²·σ1² + w2²·σ2² + 2·w1·w2·ρ·σ1·σ2 ρ = Cov(X,Y) / (σX · σY)

ρ = corrélation entre les deux actifs. Si ρ = -1, diversification maximale possible.

Risk Value at Risk (VaR)

VaR paramétrique 99% : VaR = μ - 2.33·σ·√T·Notionnel VaR 95% : z = 1.645 VaR 99% : z = 2.326

La VaR 99% sur 1 jour = perte maximale dans 99% des cas sur 1 jour. Elle ne dit pas ce qui se passe dans le 1% restant.

Oblig. Duration et sensibilité

Duration Macaulay = Σ [t · CF_t / (1+r)^t] / Prix Duration Modifiée = D_Mac / (1 + r) ΔP/P ≈ -D_mod × Δr

Duration d'un ZC = maturité. Pour une obligation, D < maturité. Convexité = dérivée seconde du prix.

Fama-French Modèle 3 facteurs

E(Ri) - Rf = βi·(Rm-Rf) + si·SMB + hi·HML SMB = Small Minus Big (taille) HML = High Minus Low (valeur/B-to-M)

Extension du CAPM. SMB capte la prime small cap, HML la prime value. Fama-French 5 facteurs ajoute RMW et CMA.

Oblig. Calcul du taux forward (simplifié)

Prix ZC : P = 100 / (1 + r)^T Taux ZC depuis prix : r = (100/P)^(1/T) - 1 Taux forward : f(1,2) = (1+r2)²/(1+r1) - 1

Un ZC ne verse aucun coupon intermédiaire — tout le flux est à maturité. Sa duration = sa maturité.

Options Payoffs standards

Long Call : max(S_T - K, 0) - Prime Long Put : max(K - S_T, 0) - Prime Straddle : max(|S_T - K|, 0) - 2×Prime Forward : S_T - F (pas de prime)

Le forward ne coûte rien à l'initiation (valeur = 0). L'option a un payoff asymétrique, le forward est linéaire.

Sharpe Ratios de performance

Sharpe = (Rp - Rf) / σp Sortino = (Rp - Rf) / σ_downside Information = (Rp - Rb) / Tracking Error

Sharpe : rendement excédentaire par unité de risque total. Sortino pénalise uniquement la vol à la baisse.

Calcul Identités remarquables

(a+b)(a-b) = a² - b² Ex: 47×53 = (50-3)(50+3) = 2500-9 = 2491 Ex: 55×45 = (50+5)(50-5) = 2500-25 = 2475 (a+b)² = a² + 2ab + b²

Indispensable pour les calculs mentaux en entretien. Toujours chercher un nombre rond au centre.

Var Swap Payoff Variance Swap

Payoff = N × (σ²_réalisée - K²_var) σ réalisée = √(252/n × Σ ln(Si/Si-1)²) Prix call ATM ratio maturité : Ratio = √(T1/T2)

N = notionnel en variance. En equity, corrélation spot-vol négative → long var swap = short delta.
